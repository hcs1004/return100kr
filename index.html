RETURN 100 서비스란?

오프라인/온라인/모바일에서 제품 구매 또는
이용 금액을 결제 후, 영수증 하단에 출력된
QR-Code를 전용앱을 통해 입력하시면
결제 금액의 30~100%를 포인트로 적립해
드리는 신개념 플렛폼 서비스입니다.

R100 포인트 적립하기

R100 포인트는 전용 앱을 이용해서만 적립가능하며, 부정한 적립을 방지하기 위해 결제 완료후 7일안에 적립을 하지 않으시면 자동 소멸됩니다.

R100 포인트

현재까지 적립된 포인트입니다.
누적된 R100 포인트는 어플이 실행되면 적립된 포인트의 1/2000씩 R100 캐쉬로 변환되니 하루에 한번은 리턴100 어플을 실행해주세요. ( 유령 회원 방지시스템 적용! )

ex) 누적 100만원 이용시
: 500원 X 10일후 = 약 5,000원 환급!

R100 캐쉬

적립된 R100 캐쉬는 전국 "RETURN 100" 온/오프라인 가맹점에서
현금처럼 사용 가능합니다. ( 1캐쉬 = 1원 )

현금 전환

2만원이상 R100 캐쉬가 적립된 경우 [ 인출신청 ] 메뉴를 이용하시면 수수료 차감후 출금가능합니다. ( 출금수수료 = 13.3% )

상품권 교환

보유한 R100 캐쉬를 이용해 원하는 상품권으로 교환 가능합니다.
단, 저희 R100 서비스 플랫폼에서는 상품권을 직접 판매하지 않으며
보유하신 캐쉬를 제휴사의 상품권과 교환 후 보내 드리고 있습니다.
